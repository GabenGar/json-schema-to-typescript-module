# json-schema-to-typescript-module

## Table of contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

Install dependencies:

```sh
npm install
```

## Usage

Build the project:

```sh
npm run build
```

Start the result:

```sh
npm start
```

## Development

Check files:

```sh
npm run check
```

Format files:

```sh
npm run format
```
